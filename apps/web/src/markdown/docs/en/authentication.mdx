# Authentication

To use Trans, you need to authenticate with our Engine, it only takes a few seconds to get started.

---

## Sign in

You can sign in directly on [Trans Dashboard](https://trans.ai/login) or run the following command:

```bash title="Terminal"
npx trans@latest auth login
```


If you already have an account, you will get automatically signed in and a session will be saved on your machine.

---

## Sign out

You can sign out by running the following command:

```bash title="Terminal"
npx trans@latest auth logout
```

---

## User session

You can check if you are signed in by running the following command:

```bash title="Terminal"
npx trans@latest auth whoami
```

---


# Environment Variables

You can set the following environment variables to configure Trans:

- `TRANS_API_KEY`: The API key to use for authentication.
- `TRANS_PROJECT_ID`: The project ID to use for the project.

You find them in the [Trans Dashboard](https://trans.ai/login).
